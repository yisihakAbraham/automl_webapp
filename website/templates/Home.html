Machine learning Models

In recent years there has been a rise in the development of AI and one of a fundamental concept when it comes to AI is machine learning. Machine learning can be described as the ability of a computer to learn without being explicitly programmed. Although we can get huge benefit from using machine learning, the process of creating one can be repetitive and difficult. It is time and resource consuming, the concept can be complex to understand and it requires programming knowledge. In order to solve these problems we came up with this project idea, a web-based application that provides user with a pre-built machine learning models.
